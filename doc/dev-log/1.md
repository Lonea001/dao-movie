#### 后端接口设计

FastAPI

​	APIRouter	- 	路由分组

​	Query	       -	对 URL 做校验

​	Depends	  -	注入数据库连接、当前用户、权限校验

​	Session	    - 	数据库会话，做增删改查

​	Optional	 -	  查询参数可选

​	or_		  -	   SQL 里的 or  查询

api

​	<font color='green'>get_movies</font>

​		arguments	-->

​			page		-->	1

​			page_size	-->	1-20

​			q	  		-->	Optional[str]	->  Query(None)

​			genre	      -->	Optional[str]

​			year		-->	 Optional[str]

​			sort_by	  str	-->	"rating"    rating / year / rating_count

​			db		Session



​		默认访问：

​			GET /movies

​			`page=1`

​			`page_size=20`

​			`q=None`

​			`genre=None`

​			`year=None`

​			`sort_by="rating"`

​		用户查询参数：

```
GET /movies?keyword=周星驰&page=2&page_size=10

	q = "周星驰"

	page_size = 10
```
