电影数据来自 github 上找到的数据集。

### 后端

后端处理这些功能：

- 电影基础信息
- 榜单关系
- 互动记录
- 管理员账号存进数据库

#### 电影模块

已经支持：

- 电影列表中的分页
- 关键字搜索
- 类型筛选
- 年份筛选
- 排序
- 类型列表
- Top 电影
- 详情页



#### 榜单模块

- 榜单列表
- 榜单详情
- 创建
- 更新
- 删除



#### 管理员模块

- 登录
- token 校验



#### 互动模块

- 点赞
- 收藏
- 统计查询



#### 模型层

- Movie
- Ranking
- RankingItem
- Interaction
- Admin



### 前端

前端围绕接口做展示和管理

路由已经拆成：

- 首页
- 电影列表
- 电影详情
- 榜单列表
- 榜单详情
- 后台 
  - /admin               登录页
  - /admin/home    管理页

前端 Vite 用代理把**接口请求** /api 转到 8000端口 后端服务上
